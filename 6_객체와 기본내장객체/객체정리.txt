* 객체 : JS가 명령 가능한 모든 대상
1. 기본 내장 객체 : 컴퓨터 언어가 갖고 있는 기본적인 객체 - 첫글자 대문자로 작성
    1) String : 문자열 객체
    2) Number : 숫자 객체
    3) Array : 배열 객체
    4) Math : 수학 객체
    5) Date : 날짜 시간 객체
    6) Object : 사용자 정의 객체
2. BOM(browser object model) : 브라우저 관련 객체 - 소문자로 작성
    1) window : 창 객체
    2) location : 주소표시줄 객체
    3) history : 주소내역 객체
    4) screen : 운영체제 화면 객체
    5) navigator : 브라우저 정보 객체 
    6) document : 문서객체
3. DOM(document object model ) : 문서객체모델
    1) 태그
    2) 태그의 속성
    3) 텍스트
    4) 태그의 이벤트 등등
